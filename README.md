# BackEnd

| Variable | Description |
| ------ | ------ |
| REACT_REDUX_FRONT_URI | Current url of the app front |
| MONGODB_URI | URI of connection to mongodb |

### Install BackEnd dependecies
With NPM:
```sh
$ npm i
```
or Yarn:
```sh
$ yarn
```
### Run BackEnd App
With NPM:
```sh
$ npm start
```
or Yarn:
```sh
$ yarn start
```
